论坛整合管理
开启论坛整合时设置本页内容，一般情况下请不要修改，修改前请备份 api/api_config.php 文件，以防止修改错误导致站点无法运行
是否开启论坛整合: 否 > 是
如果选择“是”开启整合，请正确设置一下各项内容，设置不正确可能会导致程序无法运行
选择要整合的用户中心: UCenter > phpWind 论坛
UCenter 应用 ID:
该值为当前站点在 UCenter 的应用 ID，一般情况请不要改动
UCenter 通信密钥:
通信密钥用于在 UCenter 和 人才网 之间传输信息的加密，可包含任何字母及数字，请在 UCenter 与 人才网 设置完全相同的通讯密钥，以确保两套系统能够正常通信
UCenter 访问地址:
如果你的 UCenter 访问地址发生了改变，请修改此项。不正确的设置可能导致站点功能异常，请小心修改。 格式: http://www.sitename.com/uc_server (最后不要加'/')
UCenter IP 地址:
如果你的服务器无法通过域名访问 UCenter，可以输入 UCenter 服务器的 IP 地址')
UCenter 数据库服务器:
可以是本地也可以是远程数据库服务器，如果 MySQL 端口不是默认的 3306，请填写如下形式：127.0.0.1:6033
